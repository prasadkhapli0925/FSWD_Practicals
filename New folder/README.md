# React Calculator Application

A simple, elegant calculator application built with React JS. It features a modern UI with support for basic arithmetic operations.

## Features

✨ **Basic Operations**: Addition, Subtraction, Multiplication, Division
✨ **Additional Functions**: Percentage, Sign toggle (+/-), Decimal support
✨ **Responsive Design**: Works on desktop and mobile devices
✨ **Modern UI**: Beautiful gradient background and smooth animations
✨ **Clear Display**: Large, easy-to-read display with green text on dark background

## Installation

1. Navigate to the project directory:
   ```bash
   cd calculator
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

The calculator will open automatically in your browser at `http://localhost:3000`

## Available Scripts

### `npm start`
Runs the app in development mode. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### `npm build`
Builds the app for production to the `build` folder.

### `npm test`
Launches the test runner in interactive watch mode.

## How to Use

1. **Number Entry**: Click number buttons (0-9) to enter values
2. **Operations**: Click an operation button (+, -, ×, ÷) to perform calculations
3. **Decimal**: Click the decimal (.) button to add decimal points
4. **Equals**: Click (=) to see the result
5. **Clear**: Click AC to clear the display and reset
6. **Sign Toggle**: Click +/- to toggle between positive and negative
7. **Percentage**: Click % to convert to percentage
