# Quick Start Guide - Tic Tac Toe Game

Get the Tic Tac Toe game running in minutes!

## ⚡ Quick Setup

1. **Open terminal in this directory**:
   ```bash
   cd tic-tac-toe
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the game**:
   ```bash
   npm start
   ```

4. **Open your browser** to `http://localhost:3000` and start playing!

## 🎮 First Game

- Player X goes first
- Click on any square to make a move
- First to get 3 in a row wins
- Click "New Game" to restart

## 📋 Game Rules

✅ Get 3 symbols in a row (horizontal, vertical, or diagonal)
✅ Players take turns (X and O)
✅ Cannot click on already filled squares
✅ Game ends when someone wins or board is full

That's it! Enjoy the game 🎉

## 🆘 Troubleshooting

**Port 3000 is in use?**
```bash
# Use a different port
PORT=3001 npm start
```

**Dependencies not working?**
```bash
# Clean and reinstall
rm -rf node_modules
npm install
```
