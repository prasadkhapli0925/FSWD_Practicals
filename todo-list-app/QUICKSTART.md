# Quick Start Guide - To-Do List App

Get your To-Do List application running in minutes!

## ⚡ Quick Setup

1. **Open terminal in this directory**:
   ```bash
   cd todo-list-app
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the app**:
   ```bash
   npm start
   ```

4. **Open your browser** to `http://localhost:3000` and start managing tasks!

## 🎯 First Steps

### Adding a Task
1. Type in the input field at the top
2. Click "➕ Add Task" or press Enter
3. Your task appears in the list!

### Managing Tasks
- ✅ **Check box** = Mark complete
- ✏️ **Edit icon** = Modify task
- 🗑️ **Delete icon** = Remove task

### Filtering
- **All Tasks** = Show everything
- **Pending** = Only incomplete tasks
- **Completed** = Only finished tasks

## 📊 Key Features

✅ Add unlimited tasks

✅ Beautiful animations

## 🆘 Troubleshooting

**Port 3000 in use?**
```bash
PORT=3001 npm start
```

**Dependencies issues?**
```bash
rm -rf node_modules
npm install
```

**Need to restart?**
Press `Ctrl+C` in terminal, then `npm start` again
